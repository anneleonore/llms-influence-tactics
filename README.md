## LLMs
We compared the performance of four open-source LLms, in particular:
(1) Llama-3.1-8B (Meta AI, 2024)

(3) Mistral-7B-Instruct-v0.3 (Jiang et al., 2023)
(4) Qwen2.5-3B-Instruct (Qwen Team, 2024)
